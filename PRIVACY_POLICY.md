# 개인정보처리방침

**최종 수정일: 2026년 2월 25일**

본 개인정보처리방침은 **fullweight**(이하 "앱")의 개인정보 처리에 관한 사항을 안내합니다.

---

## 1. 개인정보의 수집 항목 및 수집 방법

### 가. 사용자가 직접 입력하는 정보

앱은 다음과 같은 정보를 사용자의 기기(로컬)에만 저장하며, 별도의 서버에 수집하지 않습니다.

| 구분         | 항목                                                                                          |
| ------------ | --------------------------------------------------------------------------------------------- |
| 프로필 정보  | 키(cm), 생년월일, 성별, 운동 빈도·시간·강도                                                   |
| 신체 기록    | 몸무게(kg), 허리둘레(cm), 골격근량(kg), 체지방률(%), 체지방량(kg), 운동 여부, 음주 여부, 메모 |
| 식사 기록    | 식사 종류(아침/점심/저녁/간식), 음식 설명, 칼로리, 탄수화물, 단백질, 지방                     |
| 사진         | 눈바디(체형 변화) 사진, 음식 사진                                                             |
| 챌린지 기록  | 시작/목표 체중·골격근량·체지방률, 기간, 달성 여부                                             |
| 앱 잠금 설정 | 4자리 PIN 번호                                                                                |

### 나. 자동으로 수집되는 정보

앱 자체적으로 자동 수집하는 개인정보는 없습니다. 다만, 앱에 포함된 제3자 SDK(광고, 결제)가 기기 식별자 및 광고 ID 등을 수집할 수 있으며, 이는 해당 서비스의 개인정보처리방침에 따릅니다.

---

## 2. 개인정보의 이용 목적

| 목적              | 설명                                                                      |
| ----------------- | ------------------------------------------------------------------------- |
| 체중·식사 관리    | 사용자의 건강 기록 관리 및 통계 표시                                      |
| AI 음식 분석      | 음식 사진을 OpenAI API에 전송하여 영양소 자동 분석 (사용자 요청 시에만)   |
| Google Drive 백업 | 사용자가 직접 로그인 및 백업을 실행한 경우에만 Google Drive에 데이터 저장 |
| 인앱 결제         | RevenueCat을 통한 구독 및 구매 상태 관리                                  |
| 광고 표시         | Google AdMob을 통한 배너 및 전면 광고 게재                                |

---

## 3. 개인정보의 보관 및 저장

- **모든 사용자 데이터는 기기 내부(로컬)에만 저장**됩니다.
- 앱 개발자의 별도 서버에 사용자 데이터를 저장하거나 수집하지 않습니다.
- 앱을 삭제하면 기기에 저장된 모든 데이터가 함께 삭제됩니다.
- Google Drive 백업 데이터는 사용자의 Google 계정에 저장되며, 사용자가 직접 관리할 수 있습니다.

---

## 4. 제3자 서비스 제공

앱은 다음 제3자 서비스를 사용하며, 각 서비스의 개인정보처리방침에 따라 정보가 처리될 수 있습니다.

| 서비스           | 제공자           | 목적                | 전송되는 정보                  | 개인정보처리방침                                                            |
| ---------------- | ---------------- | ------------------- | ------------------------------ | --------------------------------------------------------------------------- |
| Google AdMob     | Google LLC       | 광고 게재           | 기기 식별자, 광고 ID           | [Google 개인정보처리방침](https://policies.google.com/privacy)              |
| RevenueCat       | RevenueCat, Inc. | 인앱 결제 관리      | 익명 사용자 ID, 구매 내역      | [RevenueCat 개인정보처리방침](https://www.revenuecat.com/privacy)           |
| OpenAI API       | OpenAI, Inc.     | AI 음식 영양소 분석 | 음식 사진 (사용자 요청 시에만) | [OpenAI 개인정보처리방침](https://openai.com/privacy/)                      |
| Google Drive API | Google LLC       | 데이터 백업/복원    | 앱 데이터 (사용자 요청 시에만) | [Google 개인정보처리방침](https://policies.google.com/privacy)              |
| Google Sign-In   | Google LLC       | Google Drive 인증   | 이메일 주소, OAuth 토큰        | [Google 개인정보처리방침](https://policies.google.com/privacy)              |
| Open Food Facts  | Open Food Facts  | 음식 영양정보 검색  | 검색어 (음식 이름)             | [Open Food Facts 개인정보처리방침](https://world.openfoodfacts.org/privacy) |

## 5. 수집하는 권한

앱은 다음 기기 권한을 필요에 따라 요청하며, 사용자가 거부할 수 있습니다.

| 권한        | 목적                             | 요청 시점         |
| ----------- | -------------------------------- | ----------------- |
| 카메라      | 눈바디 사진 촬영, 음식 사진 촬영 | 해당 기능 사용 시 |
| 사진/미디어 | 갤러리에서 사진 선택             | 해당 기능 사용 시 |
| 파일 접근   | InBody CSV 파일 가져오기         | 해당 기능 사용 시 |
| 생체인증    | 앱 잠금 해제 (지문, 얼굴 인식)   | 앱 잠금 설정 시   |

---

## 6. 아동의 개인정보 보호

앱은 만 14세 미만의 아동을 대상으로 하지 않으며, 아동의 개인정보를 의도적으로 수집하지 않습니다. 만 14세 미만의 아동이 앱을 사용한 사실을 인지하게 되면, 해당 정보를 즉시 삭제하기 위한 조치를 취하겠습니다.

---

## 7. 개인정보의 파기

- **앱 삭제**: 앱을 삭제하면 기기에 저장된 모든 데이터가 자동으로 삭제됩니다.
- **데이터 초기화**: 앱 내 설정 > "모든 기록 삭제" 기능을 통해 언제든지 모든 데이터를 삭제할 수 있습니다.
- **Google Drive 백업 삭제**: Google Drive에 저장된 백업 데이터는 사용자가 Google Drive에서 직접 삭제할 수 있습니다.

---

## 8. 이용자의 권리

- 사용자는 언제든지 앱 내에서 자신의 데이터를 조회, 수정, 삭제할 수 있습니다.
- 사용자는 앱 내 내보내기 기능(JSON, CSV, ZIP)을 통해 자신의 데이터를 내려받을 수 있습니다.
- 사용자는 기기 설정에서 앱의 권한을 변경하거나 철회할 수 있습니다.

---

## 9. 개인정보처리방침의 변경

본 개인정보처리방침은 관련 법령 또는 서비스 변경에 따라 수정될 수 있으며, 변경 시 앱 업데이트 또는 본 페이지를 통해 안내합니다.
